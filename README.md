## ✨ Funcionalidades do Dashboard

- **Gráfico de rosca** e **barras** com hover animado

## 🚀 Como publicar no GitHub Pages

### 1. Crie o repositório
```bash
git init
git add .
git commit -m "feat: dashboard relatorio n2 canaa telecom fev2026"
git branch -M main
git remote add origin https://github.com/SEU-USUARIO/canaa-relatorio-n2.git
git push -u origin main
```

### 2. Ative o GitHub Pages
1. Acesse o repositório no GitHub
2. Vá em **Settings → Pages**
3. Em **Branch**, selecione `main` e pasta `/ (root)`
4. Clique em **Save**
5. Aguarde ~1 minuto e acesse: `https://SEU-USUARIO.github.io/canaa-relatorio-n2/`
